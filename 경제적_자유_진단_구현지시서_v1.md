## 1. 프로젝트 개요

### 진단 이름
경제적 자유 진단 (Economic Freedom Diagnosis)

### 훅
"나는 경제적 자유에 얼마나 가까운가?"

### 목적
스레드/X에서 링크 클릭 → 모바일 브라우저에서 진단 → 결과 확인 → 카페 가입 유도

- **단일 HTML 파일** (index.html 하나에 HTML + CSS + JS 모두 포함)
- **Chart.js** (CDN: https://cdn.jsdelivr.net/npm/chart.js) — 레이더 차트용
- **LocalStorage** — 응답 저장 (새로고침 시 복구)
- **Google Analytics 4** (gtag.js) — 측정 ID: G-KJNV6DHSGC
- **배포**: GitHub Pages 또는 Netlify (정적 파일)

## 3. UI/UX 스펙

### 3-1. 모바일 우선
- `max-width: 480px` 기준 설계, 데스크톱에서도 중앙 정렬 카드 형태
- 터치 영역 최소 48px
- 선택지 버튼 간 간격 12px

### 3-2. 1화면 1문항
- 한 화면에 문항 하나만 표시
- 선택지 탭 시 0.3초 후 자동으로 다음 문항으로 슬라이드
- 뒤로가기 버튼으로 이전 문항 수정 가능

### 3-3. 프로그레스 바
- 화면 상단 고정
- 전체 34문항 기준 퍼센트 표시
- 인사이트 브레이크는 프로그레스에 포함하지 않음 (34문항만 카운트)
- 색상: 골드(#C9A96E) 채움, 회색(#DEE2E6) 배경

### 3-4. 전환 애니메이션
- 문항 전환: 좌→우 슬라이드 (300ms ease-out)
- 선택지 선택 시: 테두리 색상 골드로 전환 + 체크 아이콘 표시
- 인사이트 브레이크 진입: 페이드인 (500ms)

### 3-5. 시작 화면
```
[네이비 배경]

경제적 자유 진단

"나는 경제적 자유에 얼마나 가까운가?"

34개의 질문으로 당신의 현재 위치를 진단합니다.
소요 시간: 약 5~7분

[시작하기] ← 골드 버튼

성공을 디자인하라 | BRIDGE
```

### 3-6. 축 전환 화면 (각 축 시작 전)
축 2, 3, 4 시작 전에 짧은 전환 화면을 보여준다:

- 축 2 시작 전: "정답을 맞히는 것이 아닙니다. 평소 당신이 돈에 대해 어떻게 생각하는지를 봅니다."
- 축 3 시작 전: "잘하고 있느냐가 아닙니다. 지금 어디에 서 있는지를 봅니다."
- 축 4 시작 전: "많이 배웠느냐가 아닙니다. 어떻게 배우느냐가 중요합니다."

각 전환 화면은 2초 후 자동 넘어가거나 탭으로 넘길 수 있음.

## 4. 34문항 전문

### 데이터 구조
각 문항을 아래 형태의 JavaScript 객체 배열로 구현한다:

```javascript
const questions = [
  {
    id: "P1-01",
    axis: 1,               // 축 번호 (1~4)
    axisName: "인식",
    title: "문항 제목 (내부용)",
    prompt: "문항 본문 텍스트",
    choices: [
      { label: "A", text: "선택지 텍스트", score: 1, tag: "태그명" },
      // ...
    ],
    scoring: "점수화 여부 및 방법",
    randomize: false,       // 선택지 순서 랜덤화 여부
    note: "교차 검증 등 참고 사항"
  },
  // ...
];
```

### 6-2. 5단계 등급 변환 (절대 기준)

각 축의 원점수를 비율로 변환 후 5단계로 분류:

```javascript
function getGrade(score, maxScore) {
  const ratio = score / maxScore;
  if (ratio >= 0.85) return 5;  // 매우 높음
  if (ratio >= 0.65) return 4;  // 높음
  if (ratio >= 0.45) return 3;  // 보통
  if (ratio >= 0.25) return 2;  // 낮음
  return 1;                      // 매우 낮음
}

const grade1 = getGrade(axis1Score, 12);
const grade2 = getGrade(axis2Score, 27);
```

### 6-4. 6유형 분류 알고리즘

```javascript
function classifyType(grade1, grade2, grade3, grade4) {
  // 우선순위 1: 아직 시작 전인 사람
  if (grade1 <= 2 && grade2 <= 2 && grade3 <= 2 && grade4 <= 2) {
    return "starter";
  }
  
  // 우선순위 2: 균형 잡힌 성장자
  const allGrades = [grade1, grade2, grade3, grade4];
  const minGrade = Math.min(...allGrades);
  const maxGrade = Math.max(...allGrades);
  if (minGrade >= 3 && (maxGrade - minGrade) <= 1) {
    return "balanced";
  }
  
  // 우선순위 3: 아는데 안 하는 사람 (인식/지식 높고 경험 낮음)
  if ((grade1 >= 3 || grade2 >= 3) && grade3 <= 2) {
    // 학습도 높으면 "서재에 머무는 사람"
    if (grade4 >= 3) return "bookworm";
    return "knower";
  }
  
  // 우선순위 4: 하는데 모르고 하는 사람 (경험 높고 지식/인식 낮음)
  if (grade3 >= 3 && (grade1 <= 2 || grade2 <= 2)) {
    return "doer";
  }
  
  // 우선순위 5: 패러다임만 앞서간 사람 (인식 높고 나머지 낮음)
  if (grade1 >= 4 && grade2 <= 2 && grade3 <= 2) {
    return "visionary";
  }
  
  // 우선순위 6: 열심히 배우는데 방향 없는 사람 (학습 높고 경험 낮음)
  if (grade4 >= 3 && grade3 <= 2) {
    return "bookworm";
  }
  
  // 기본값: 간극이 가장 큰 패턴으로 분류
  if (gap_perception_experience >= gap_knowledge_experience) {
    return grade1 > grade3 ? "knower" : "doer";
  } else {
    return grade2 > grade3 ? "knower" : "doer";
  }
}
```

## 7. 교차 검증 로직

교차 검증은 결과 페이지의 "자기 보고 vs 실제 비교" 섹션에서 사용한다.
가입 후 잠금 해제 시에만 표시.

### 7-1. Layer 1: Popova 연결 (핵심)
```javascript
// P1-03 감정 vs P3-08 행동 비교
const popovaGap = {
  emotion: answers["P1-03"].tag,  // 감정 반응
  action: answers["P3-08"].tag,   // 실제 행동
  gap: getPopovaGapMessage(answers["P1-03"].tag, answers["P3-08"].tag)
};

function getPopovaGapMessage(emotion, action) {
  if (emotion === "Status" && action === "무행동") 
    return "자극은 받지만 행동으로 이어지지 않는 패턴이 있습니다.";
  if (emotion === "Worship" && action === "탐색")
    return "관심은 있지만 실행까지는 가지 않는 경향이 있습니다.";
  if (emotion === "Vigilance" && action === "무행동")
    return "걱정이 행동을 막고 있을 수 있습니다.";
  if (action === "실행" || action === "관계행동")
    return "감정이 행동으로 이어지는 실행력을 가지고 있습니다.";
  return "감정과 행동 사이의 연결을 살펴볼 필요가 있습니다.";
}
```

### 7-2. Layer 4: Dunning-Kruger
```javascript
// P4-07 자기 평가 vs 축 2 실제 점수 비교
const selfAssessment = answers["P4-07"].tag;
const actualKnowledge = grade2;

function getDunningKrugerMessage(self, actual) {
  const selfLevel = { "초보": 1, "기본": 2, "중급": 3, "전문": 4, "고립": null };
  const sl = selfLevel[self];
  if (sl === null) return "주변에 대화 상대가 없어 자기 수준을 확인하기 어려운 상태입니다.";
  const gap = sl - actual;
  if (gap >= 2) return "실제 지식보다 자기 평가가 높은 편입니다. 모르는 것을 모르는 상태일 수 있습니다.";
  if (gap <= -2) return "실제 지식에 비해 자기 평가가 낮은 편입니다. 스스로를 과소평가하고 있을 수 있습니다.";
  return "자기 인식과 실제 수준이 비교적 일치합니다.";
}
```

### 결과 공유 버튼
```javascript
function shareResult() {
  const shareData = {
    title: '경제적 자유 진단 결과',
    text: `나는 "${typeInfo[resultType].name}" — ${typeInfo[resultType].metaphor}. 여정 ${typeInfo[resultType].stage}단계.\n나는 경제적 자유에 얼마나 가까운가?`,
    url: window.location.href
  };
  
  if (navigator.share) {
    navigator.share(shareData);
  } else {
    // 폴백: 클립보드 복사
    navigator.clipboard.writeText(shareData.text + '\n' + shareData.url);
    alert('결과가 복사되었습니다!');
  }
}
```
