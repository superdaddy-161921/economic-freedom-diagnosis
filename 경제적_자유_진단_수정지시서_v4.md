## 수정 1: 선택지 라벨 순서 고정 (랜덤화 버그 수정)

**문제:** 랜덤화 문항(P1-05, P1-06, P3-03, P3-06, P4-05)에서 선택지를 셔플할 때 라벨(A/B/C/D/E)도 함께 섞여서 D→A→B→C 같은 순서로 표시됨.

**원인:** 929번 줄에서 `c.label`(원래 라벨)을 그대로 표시하고 있음.

**수정:** 선택지 버튼 렌더링 시 `c.label` 대신 위치 기반 라벨을 부여.

현재 코드 (약 929번 줄):
```javascript
btn.innerHTML = `<span class="choice-label">${c.label}</span><span class="choice-text">${c.text}</span>`;
```

변경:
```javascript
const posLabels = ['A','B','C','D','E','F'];
btn.innerHTML = `<span class="choice-label">${posLabels[idx]}</span><span class="choice-text">${c.text}</span>`;
```

**주의:** 점수 계산은 `c.score`와 `c.tag`로 하므로 라벨 변경은 점수에 영향 없음. `selectChoice` 함수에서 `c.label`을 저장하는 부분(944번 줄)도 원래 라벨 그대로 저장해도 무방 (내부 기록용이므로).

## 수정 2: P2-01 시나리오 질문 자연스럽게 변경

**문제:** 친구가 자기 상황만 말하고 의견을 구하지 않아서, "뭐라고 하겠습니까?"가 뜬금없음.

**현재 (약 580번 줄):**
```
친구가 말합니다.
"나 자산이 5억이야. 대출이 3억 있긴 한데, 차도 바꾸고 싶고 여행도 가고 싶어."

당신이라면 뭐라고 하겠습니까?
```

**변경:**
```
친구가 말합니다.
"나 자산이 5억이야. 대출이 3억 있긴 한데, 이번에 차 바꾸고 해외여행도 가려고. 괜찮겠지?"

당신이라면 뭐라고 하겠습니까?
```

코드에서 해당 prompt 문자열을 찾아서 변경:

현재:
```javascript
prompt:'친구가 말합니다.\n"나 자산이 5억이야. 대출이 3억 있긴 한데, 차도 바꾸고 싶고 여행도 가고 싶어."\n\n당신이라면 뭐라고 하겠습니까?'
```

변경:
```javascript
prompt:'친구가 말합니다.\n"나 자산이 5억이야. 대출이 3억 있긴 한데, 이번에 차 바꾸고 해외여행도 가려고. 괜찮겠지?"\n\n당신이라면 뭐라고 하겠습니까?'
```

## 수정 3: P1-03 D선택지 의미 명확화

**문제:** "잘됐다. 근데 내 방식대로 하는 게 맞아."가 (a) 자기 확신인지 (b) 상대 부정인지 모호함.

**현재 (약 522번 줄):**
```javascript
{label:"D", text:"잘됐다. 근데 내 방식대로 하는 게 맞아.", tag:"자기확신"}
```

**변경:**
```javascript
{label:"D", text:"잘됐다. 나는 내 계획대로 가련다.", tag:"자기확신"}
```
