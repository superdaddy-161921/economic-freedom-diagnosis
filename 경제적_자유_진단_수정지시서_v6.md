## 수정 1: P2-05 문항 "그만뒀고" → "넣는 걸 멈췄고"

**문제:** "멈췄고"가 매도+현금 전환으로 오해될 수 있음.

**현재:**
```javascript
prompt:'두 사람이 같은 금액을 매달 투자합니다.\n첫째는 25살에 시작해서 35살에 멈췄고(10년간),\n둘째는 35살에 시작해서 65살까지 계속했습니다(30년간).\n\n65살이 됐을 때, 누가 더 많을까요?'
```

**변경:**
```javascript
prompt:'두 사람이 같은 금액을 매달 투자합니다.\n첫째는 25살에 시작해서 35살에 넣는 걸 멈췄고(10년간),\n이후 한 푼도 넣지 않고 그대로 두었습니다.\n둘째는 35살에 시작해서 65살까지 계속했습니다(30년간).\n\n65살이 됐을 때, 누가 더 많을까요?'
```

## 수정 2: "성공을 디자인하라 네이버 카페" → "성공을 디자인하라 | 네이버 카페"

**위치:** 파일 전체에서 "성공을 디자인하라 네이버 카페"를 검색하여 모두 변경.

**변경:** `성공을 디자인하라 네이버 카페` → `성공을 디자인하라 | 네이버 카페`

구분선 `|` 양쪽에 반각 공백 또는 `&nbsp;` 포함하여 가독성 확보.
